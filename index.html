Comandos para GIT

  1. git --version (para verificar que git este instalado)
  2. git init ( se crea una carpeta .git)
  3. Para configurar:
    • git config --global user.name "Nombre Apellido"
    • git config --global user.email mail@example.com
    • git status (para chequear)
  4. git add o git add . //// (git add nombreArchivo [vas agregando de a uno] ; git add . [te agrega TODO lo que tengas en la carpeta])
  5. git commit -m “El msj entre comillas”
  6. git log (chequear los commit) // git log --oneline --decorate --all --graph (historial de cambios hechos en el respositorio)
  7. git checkout y pongo el ID del commit (para movernos a un commit)
